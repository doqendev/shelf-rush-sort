# Screenshot And Layout Matrix

Automated layout guards cover the source-of-truth screenshot widths:

- 320 px phone
- 360 px phone
- 390 px phone
- 430 px phone
- tablet width

The layout test verifies the 5x3 rack, all 45 front cells, optional moving-lane
space, non-overlapping targets, minimum target size, and hit-test routing at
each width.

Manual screenshot capture is still required for milestone builds using the
manual playtest checklist states: level start, hidden previews, horizontal and
vertical lanes, blockers, selected product, invalid feedback, clear/reveal
frames, win/loss panels, shop, and settings.
